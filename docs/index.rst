LayeredCompModel
================

Hierarchical tree-based regressor for robust predictions (e.g., parcel sale prices) using weighted Wilson score intervals.

**Scikit-learn compatible.**

Installation
------------

.. code-block:: bash

   pip install layeredcompmodel
